**Bài 4:** Giả sử gán các biến với các giá trị như bên dưới

 ```sh
 with = 17
 height = 12.0
 ```

Với mỗi biểu thức dưới đây, hãy điền giá trị và kiểu của chúng 

 ```sh
 1. with / 2
 2. with / 2.0
 3. height / 3
 4. 1 + 2 + 5
 ```

Sử dụng trình biên dịch của python (IDLE) để kiểm tra lại câu hỏi của bạn.
